Python爬虫构建

关于爬虫

本质是一个访问网站，提取某些信息的程序。

“爬取”这个动词描述获取数据的过程。其实比起爬取，爬虫的行为更像是递交一个申请表，申请到一本html文件，然后对着这本html文件写了一篇摘要呈现给用户。

理论上说，人来通过浏览器手动记录数据本质和爬虫一样，但爬虫效率高，容易把服务器爬出问题。

关于我这个爬虫

用来爬ngabbs.com的某些版面帖子标题，用于数据分析，绝对不是干什么坏事。
